# 월세 비교 계산기

단순 월세뿐 아니라 **보증금 기회비용·관리비·교통비**를 포함한 **실질 월비용**과 **월 통근시간**을 한눈에 비교하는 대시보드.

## 핵심 공식

```
실질 월비용 = 월세 + (보증금 × 연이율 ÷ 12) + 관리비 + (교통비 편도 × 2 × 출근일수 ÷ 10000)
월 통근시간 = 편도 통근시간(분) × 2 × 출근일수 ÷ 60
```

- 연이율 기본값: 3.5%
- 출근일수 기본값: 22일
- 비용과 시간은 별도 표시 (시간을 돈으로 환산하지 않음)

## 기능

- 매물 추가/삭제
- 비교 요약 카드 (최저 비용 하이라이트)
- 비용 구성 Stacked Bar 차트
- 비용 vs 통근시간 Scatter 차트
- 상세 비교 테이블
- localStorage 자동 저장/복원
- 모바일 반응형
- 토스앱 다크 테마

## 사용법

브라우저에서 `rent-compare-dashboard.html` 파일을 직접 열면 됩니다.
